# Web Starter

### Starter for web back end based on spring boot

## Includes

* Domain-starter
    * postgresql
    * ready to use basic entities for user management
    * testcontainers-postgresql
* Swagger-ui
* Register / Login / Change password / Delete user
* JWT authentication
* Job scheduling
* Testing with JUnit 5


## Stack
* Java 17
* Spring Boot 3.2.4
* Maven
* Postgres
* Docker
